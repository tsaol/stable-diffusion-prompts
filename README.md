## 咒语
### 咒语的结构
主体+场景+质量风格+照明+色彩+情绪+构图
主体：图像的主要对象，人或者物。
场景：背景或环境。
风格：风格。
照明：图像的照明条件。
色彩：主要颜色。
情绪：情绪或氛围。
构图：构图或布局。
